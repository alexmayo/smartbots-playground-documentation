# Rules & terms

This page contains the Bot Store publication rules and terms.

## Product (script) name

* Has to be descriptive
* Can not contain brand or author name
* Has to conform with all applicable Second Life Terms

## Product description

* Has to describe the product
* Can not start with product name
* Can not start with brand or author name ("start" means first 7 words)
* Has to conform with all applicable Second Life Terms

## Product image(s)

Please refer to the [Image Guidelines](./Images.md) page.

# Moderation

SmartBots Bot Store moderation team reserves the right to adjust any Store product details without notice if they do not conform with these rules and terms.
